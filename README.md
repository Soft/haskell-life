## Conway's Game of Life implemented in Haskell using SDL

    Flags:
      -h Display this help
    Keys:
      Space: Step simulation
      Backspace: Previous state
      Return: Start simulation
      Plus: Increase speed
      Minus: Decrease speed
      c: Clear the grid
      t: Toggle between infinite and bounded grid
      q: Quit
